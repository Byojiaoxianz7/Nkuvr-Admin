# 实验室预约管理系统

## 需求分析

该系统的用户分为两种：管理员、学生

以下针对两种用户分别描述其功能需求

### 1、管理员功能需求描述

- 登录
- 密码重置
- 用户管理
- 预约管理
- 实验室管理
- 退出登录 

### 2、学生功能需求描述

- 登录
- 密码重置
- 查看个人信息
- 查看实验室列表
- 预约
- 管理预约信息
- 退出登录
